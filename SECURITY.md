# Security Policy

## Supported Versions
We aim to provide security fixes for the last two minor versions:

- 0.1.x — security fixes
- 0.0.x — no longer supported

> Note: As the project is young, this matrix may evolve. Please check releases and tags.

## Reporting a Vulnerability
Please do NOT disclose security issues publicly via GitHub issues or pull requests.

- Report confidentially via email: info@rothaarsystems.de
- Alternatively, use GitHub Security Advisories (private report):
  https://github.com/sneakyx/laravel-dynamic-encryption/security/advisories

Please include (as far as possible without exposing secrets):
- Affected version(s), PHP and Laravel versions
- Steps to reproduce or proof of concept
- Impact/severity assessment
